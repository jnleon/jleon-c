In OOP, a collection is grouping of some variable number of data items (possibly zero) that have some shared significance to the problem being solved and need to be operated upon together in some controlled fashion. There are various data structures in the collections library.

In OOP, a data structure is a collection of values, the relationships among the values, and the operations that can be performed on the values

In OOP, a list is a type of data structure in the collections library that represents a countable number of ordered values, where the same value may occur more than once.
