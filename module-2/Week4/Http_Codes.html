Summary of HTTP Methods for RESTful APIs

Method CRUD Details Success Response
GET Read Get a resource 200 (OK) and requested resource
POST Create Create a new object 201 (Created), ‘Location’ header with link to /users/{id} containing new ID.
PUT Update/Replace Update an object or replace an existing object with a new one 200 (OK) if you're returning the new object or 204 (No Content)
DELETE Delete Delete an object 204 (No Content)

404 (Not Found), 400 (Bad Request) and 500 (Internal Server Error) are some common error response codes.
